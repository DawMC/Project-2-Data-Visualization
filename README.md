### **Dataset Name:** Students Performance Dataset  
### **Source:** https://www.kaggle.com/datasets/spscientist/students-performance-in-exams  
### **Number of Records:** 1,000 rows (>100 required)

### **Description:**  
This dataset includes exam scores (math, reading, writing) and demographic attributes such as gender, race/ethnicity, parental education, lunch type, and test preparation status. It is widely used for understanding performance patterns and exploring equity and achievement gaps.

### **Ethics Note:**  
This dataset contains information about **real students**, so results must be interpreted cautiously.  
Performance outcomes may be influenced by structural, cultural, or socioeconomic factors that are not captured in the dataset.  
The visualizations are **descriptive only** and should not be used to judge individual students or make broad generalizations about demographic groups.
